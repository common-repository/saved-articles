== Description ==

Adds strength to authorized users in the beginning of the article.
After clicking on it the link to the article is added to the widget on the admin panel of the user, where he can go to the link later.

== Installation ==

1. Upload the `myplugin` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
